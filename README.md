# Electric Vehicle Analysis

## Overview

Developed an interactive Tableau-based analytics dashboard to analyze electric vehicle (EV) adoption trends, vehicle performance, regional distribution, and market growth patterns. The project transforms raw EV data into actionable insights to support data-driven decision-making in the rapidly growing electric mobility sector.

## Tools & Technologies

* Tableau
* Microsoft Excel
* Data Visualization
* Business Intelligence
* Exploratory Data Analysis (EDA)

## Key Features

* Built interactive Tableau dashboards to visualize EV adoption trends across different years.
* Analyzed electric vehicle categories, manufacturers, and model performance.
* Created geographical visualizations to study regional EV distribution and growth.
* Implemented interactive filters and drill-down capabilities for detailed exploration.
* Identified trends influencing EV popularity and market expansion.

## Dashboard Insights

* Evaluated year-over-year growth in electric vehicle adoption.
* Compared performance across EV types, manufacturers, and models.
* Analyzed regional adoption patterns and geographical concentration of EV usage.
* Identified high-growth segments contributing to EV market expansion.
* Explored factors driving consumer adoption and market demand.

## Business Value

* Provided a centralized dashboard for monitoring EV market trends.
* Enabled faster analysis of adoption patterns and growth opportunities.
* Supported strategic planning through data-driven insights.
* Improved understanding of regional and manufacturer-level performance.
